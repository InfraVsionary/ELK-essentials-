# ELK Stack 

> 🚀 A clean, ready-to-run ELK (Elasticsearch, Logstash, Kibana) setup with Docker Compose.

## ✨ Features
- 📂 **stack/** → Configs (Elasticsearch, Logstash, Filebeat)  
- 🛠 **scripts/** → Maintenance helpers 
- 📖 **docs/** → Ops notes  
- 🐳 **docker-compose.yml** → Modular services
